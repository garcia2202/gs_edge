Descrição do problema abordado:

O problema de saúde abordado por este projeto refere-se à gestão eficiente do uso de medicamentos por parte dos pacientes. 
Muitas vezes, os pacientes têm dificuldades em lembrar-se de tomar seus medicamentos no horário correto, ou podem enfrentar desafios para controlar seu histórico de ingestão de medicamentos. 
Isso pode levar a consequências sérias, como a falta de eficácia do tratamento, efeitos colaterais indesejados ou mesmo complicações de saúde mais graves.

Proposta:

A solução proposta é uma "Pílula Digital" que inclui um dispositivo eletrônico acoplado às embalagens de medicamentos. 
Esse dispositivo monitora o uso dos medicamentos pelos pacientes e registra dados relevantes. 
O projeto também inclui um sistema de controle de estoque para garantir o fornecimento contínuo de medicamentos.
